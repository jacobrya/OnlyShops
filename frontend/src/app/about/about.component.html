О нас

Это финальный проект нашей команды на курсе Web-Dev 2025 KBTU. Бэкенд было написано Даниалом и Жәнібеков, а фронтенд — Нұржалғасом.

Наша команда

  • Даниал Ибрашов — GitHub: w0ikid
  • Дарханұлы Жәнібек — GitHub: janibekDarkhanuly
  • Қожамсейт Нұржалғас — GitHub: ayayaakasvin
  • Гитхаб проекта — GitHub: jacobrya

Проект написан с использованием Angular для фронтенда и Django для бэкенда. Мы использовали PostgreSQL в качестве базы данных и реализовали RESTful API для взаимодействия между клиентом и сервером. Мы также использовали JWT для аутентификации пользователей и обеспечения безопасности приложения.
